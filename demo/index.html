Detection kernels patch resource surfaces (setTimeout, EventTarget, MutationObserver, AbortController, DOM detachment) and attribute every acquisition to the lite-signal owner tree. Resources acquired outside any owner warn in real time; resources acquired inside an owner are auto-released on dispose -- no warning. Kernels install on entering this scene and uninstall on leaving it (disposer API).

The packaged component: createDashboardModel() (ghost-safe -- exactly 2 signals, zero per-event churn) + createDashboard() (pre-allocated row pool, textContent-only writes, dirty-flag rendering behind a power-of-2 frame mask). Fire an event storm: 500 events cost 500 boolean writes and one render.
